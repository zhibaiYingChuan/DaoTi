**Abstract**: We report the discovery of a physical principle in deep learning models—the Core Mapping Degeneracy Theorem and its profound parallel with gauge field theory in physics. Through systematic experiments (V34–V91) on a deep learning model for the I Ching symbolic system, we find that: (1) the projection matrix (text_proj) is not an optimization attractor but a **degenerate ground state**—the retrieval loss is nearly flat in the text_proj direction (F(d)∝d^0) with no recovery after perturbation (recovery rate=-0.4%); (2) there exist **multiple equivalent ground states**—starting from random initialization, text_proj converges to solutions completely different from the original (CKA=0.267) yet all achieving 100% retrieval accuracy; (3) the prototype vectors (gua_prototype) act as a **compensation field**—regardless of how text_proj changes, retraining gua_prototype almost fully restores retrieval performance (98.4% even after σ=10.0 perturbation); (4) task difficulty cannot break the degenerate flatness—even with softmax temperature down to 0.02, text_proj remains highly consistent with the original solution (CKA>0.96). ## 1 引言

### 1.1 从不动点到退化基态

深度学习的迁移学习长期面临一个核心矛盾：全量微调可以适应新领域，但会破坏已学到的通用知识（灾难性遗忘 [1]）；冻结基座可以保留通用知识，但限制了新领域的适应能力。近年来，LoRA [2]、Prompt Tuning [3] 等方法试图在两者之间寻找平衡，但尚未从理论上阐明何时、为何、何种参数可以被安全地冻结。

本研究始于一个看似简单的观察：在易经符号系统的深度学习模型中，投影矩阵（text_proj）在冻结条件下始终不变。最初，我们将此解释为"不动点"——text_proj是损失景观的极小值，任何偏离都会被优化过程推回。

然而，随着实验的深入，我们发现这个解释是**根本性错误的**。text_proj不是吸引子，而是**退化基态**——检索损失在text_proj方向上近乎平坦，无论text_proj取什么值，检索效果都差不多。这个发现打开了一扇通往更深层次原理的大门：深度学习系统中存在与物理学规范场论精确平行的结构。

### 1.2 核心贡献

本文的核心贡献是发现并验证了**核心映射退化定理**及其与规范场论的平行，包含四条性质：

1. **冻结不变性**：text_proj在冻结条件下严格不变（max_diff=0.0）
2. **平坦退化性**：检索损失在text_proj方向上近乎平坦（F(d)∝d^0）
3. **扰动不回归**：text_proj被扰动后不会回归到原解（回归率=-0.4%）
4. **基态多重性**：存在多个等价的text_proj解（CKA=0.267, top1=100%）

以及一个核心对应：

5. **规范场对应**：gua_prototype充当补偿场，维持检索性能的规范不变性

## 2 相关工作

### 2.1 参数高效微调（PEFT）

LoRA [2] 假设权重更新是低秩的，通过训练低秩适配矩阵来近似全量微调。我们的发现更根本：**投影矩阵根本不需要适配**——不是因为它是最优的，而是因为它是退化的（任何值都给出相同效果）。适配只需要重新定位原型向量，而原型向量充当了"补偿场"的角色。

### 2.2 Neural Collapse

Neural Collapse [5] 发现分类器的权重和类中心在训练后期收敛到等角紧框架（ETF）。我们的发现在两个维度上深化了Neural Collapse：（1）投影层的退化性——不仅是分类器，连中间投影都可以是退化的；（2）规范场结构——分类器/原型向量的适配能力构成了维持退化不变性的"补偿场"。

### 2.3 损失景观的平坦方向

近期研究表明，神经网络的损失景观存在大量平坦方向 [9,10]。我们的工作将此观察推进到了一个新层次：平坦方向不仅是几何性质，更是**规范对称性**的体现——平坦方向对应于规范变换的自由度，而补偿场的存在使得这些自由度不影响物理可观测。

### 2.4 Goldstone定理与自发对称性破缺

Goldstone定理 [11] 指出，连续对称性的自发破缺必然产生无能隙激发（Goldstone玻色子）。我们的系统展现了精确的平行：text_proj参数空间的连续对称性（多个等价解）导致了检索损失在text_proj方向上的零曲率（无能隙），而gua_prototype的领域特化则对应于对称性的自发破缺。

## 3 方法

### 3.1 模型架构

设模型 $M = \langle E, L, P, G, H \rangle$，其中：

- $E: \mathcal{V}^* \to \mathbb{R}^d$：文本编码器（2层TransformerEncoder，4头注意力）
- $L: \mathbb{R}^{d+s} \to \mathbb{R}^s$：洛书阶梯网络（6层阶梯，T=7时间步）
- $P: \mathbb{R}^d \to \mathbb{R}^s$：投影矩阵（text_proj，$d=128, s=176$）——**物质场**
- $G = \{g_1, \ldots, g_N\} \subset \mathbb{R}^s$：原型向量集（gua_prototype，$N=64$）——**规范场**
- $H: \mathbb{R}^s \to \mathcal{T}$：输出头（8任务分类）

总参数量：5,059,040。预训练语料：45.9M字符古典文本 + 138,240条合成数据。

### 3.2 检索机制与规范不变性

给定文本 $t$，检索目标卦象 $\hat{i}$：

$$\hat{i} = \arg\max_{i} \cos\big(\text{norm}(P \cdot E(t)),\ \text{norm}(g_i)\big)$$

**规范不变性**：当text_proj发生变换 $P \to P + \delta P$ 时，重训gua_prototype $G \to G^*(P + \delta P)$ 后，检索性能不变：

$$\text{top1}(P + \delta P, G^*(P + \delta P)) = \text{top1}(P, G)$$

### 3.3 道-器分离定义

**道体（Dao Ti）** $\mathcal{D} = \langle E, L, P, H \rangle$：冻结不可训练——**物质场**。  
**用体（Yong Ti）** $\mathcal{Y} = G$：可训练的适配参数（11,264参数，0.22%）——**规范场**。

## 5 理论分析

### 5.1 核心映射退化定理

**定理**：设系统 $S$ 在目标 $\mathcal{L}$ 下达到最优 $\theta^*$。则 $\theta^*$ 的核心组件 $\theta_c$（text_proj）满足：

1. **冻结不变性**：当 $\theta_c$ 被冻结时，$\theta_c^{(t)} = \theta_c^*$（严格不动点）
2. **平坦退化性**：$\nabla_{\theta_c} \mathcal{L}(\theta_c) \approx 0$ 在 $\theta_c$ 的整个参数空间内成立
3. **扰动不回归**：当 $\theta_c$ 被扰动偏离 $\theta_c^*$ 后，仅用 $\mathcal{L}$ 训练不会回归
4. **基态多重性**：存在 $\theta_c' \neq \theta_c^*$ 使得 $\mathcal{L}(\theta_c', \theta_a^*) = \mathcal{L}(\theta_c^*, \theta_a^*)$

**实验验证**：

| 性质 | 验证实验 | 关键数据 |
|------|---------|---------|
| 冻结不变性 | V60 | max_diff=0.0, CKA=1.0 |
| 平坦退化性 | V90+V91 | F(d)∝d^0, 梯度恒定~0.02 |
| 扰动不回归 | V90 | 回归率=-0.4% |
| 基态多重性 | V91 Exp3 | CKA=0.267, top1=100% |

### 5.2 与Goldstone定理的平行

| Goldstone定理 | 退化定理 |
|--------------|---------|
| 连续对称群 $G$ | text_proj参数空间的连续变换群 |
| 对称性自发破缺 | gua_prototype的领域特化 |
| Goldstone玻色子 | text_proj方向的零模式 |
| 无能隙激发 | 检索损失的零曲率 |
| 退化基态 | 多个等价text_proj |
| 真空期望值 | 特定的P*（训练随机性选择） |

**关键洞察**：P*不是"最优"的text_proj，而是由训练随机性从无数等价解中"随机选择"的一个。这就像自发对称性破缺中选择了一个特定的真空。

### 5.3 规范场对应

**补偿场假设**：存在映射 $\Delta: \delta\theta_c \mapsto \delta\theta_a$，使得：

$$\mathcal{L}(\theta_c + \delta\theta_c, \theta_a + \Delta(\delta\theta_c)) = \mathcal{L}(\theta_c, \theta_a)$$

| 规范场论 | 道-器系统 |
|---------|---------|
| 物质场 $\psi$ | text_proj（核心映射） |
| 规范场 $A_\mu$ | gua_prototype（原型向量） |
| 规范变换 | text_proj参数变换 |
| 补偿变换 | gua_prototype适配 |
| 规范不变量 | 检索精度 top1 |
| 协变导数 | $D = P + \Delta(G)$ |

**规范场方程**：

$$\frac{\partial \mathcal{L}}{\partial \theta_{a}} = 0 \quad \Rightarrow \quad \theta_a = \theta_a^*(\theta_c)$$

gua_prototype总是被优化到与当前text_proj匹配的状态——这是"补偿场"的动力学方程。

**守恒律**：

$$\frac{d}{dt}\text{top1}(\theta_c(t), \theta_a^*(\theta_c(t))) = 0$$

当gua_prototype完美补偿text_proj的变化时，检索性能守恒。

**实验验证**：V91 Exp1中，即使text_proj被扰动到σ=10.0（距离P*达141.3），重训gua_prototype后仍恢复到98.4% top1。gua_prototype的补偿能力几乎是无限的。

### 5.4 推论

**推论1（数据效率）**：$\frac{|\mathcal{D}_{\text{adapt}}|}{|\mathcal{D}_{\text{pretrain}}|} = 0.72\%$（138倍提升）

**推论2（参数效率）**：$\frac{|\mathcal{Y}|}{|M|} = 0.22\%$

**推论3（规范不变性）**：$\text{top1}(P + \delta P, G^*(P + \delta P)) \geq 98.4\%$，对任意 $\|\delta P\| \leq 10\|P\|$

**推论4（基态等价性）**：任意text_proj初始化 $\theta_c^{\text{rand}}$，训练后 $\text{top1}(\theta_c^{\text{rand}}, G^*(\theta_c^{\text{rand}})) = 100\%$

## 6 讨论

### 6.1 "大道殊途同归"的规范场论解读

我们的发现为"大道殊途同归"提供了精确的规范场论解读：

- **道（Dao）= 规范等价类**：不是某个特定的text_proj值，而是所有给出相同检索性能的text_proj的集合
- **殊途（Different Paths）= 规范变换**：不同的text_proj参数值（CKA=0.267，完全不同的解）
- **同归（Same Destination）= 规范不变量**：检索性能100%（所有等价解给出相同结果）

### 6.2 退化基态 vs 吸引子：为什么区分至关重要

将text_proj误认为"吸引子"会导致错误的预测：

| 预测 | 吸引子假说 | 退化基态假说 | 实验结果 |
|------|-----------|------------|---------|
| 扰动后回归 | ✓ | ✗ | **✗ (V90: -0.4%)** |
| F(d)∝d | ✓ | ✗ (F∝d^0) | **✗ (V91: d^-0.006)** |
| 唯一最优解 | ✓ | ✗ (多等价解) | **✗ (V91: CKA=0.267)** |
| 任务硬度打破 | 可能 | ✗ | **✗ (V91: CKA>0.96)** |
| 规范场结构 | 无 | ✓ | **✓ (V91: gp补偿)** |

退化基态假说完美预测了所有实验结果，而吸引子假说在每一项上都失败。

### 6.3 规范场论视角的深层含义

**对迁移学习的含义**：全量微调之所以导致负迁移，不是因为它"破坏了最优解"，而是因为它**破坏了规范对称性**——试图改变物质场（text_proj），而正确的做法是只适配规范场（gua_prototype）。

**对神经网络可解释性的含义**：规范场论提供了一种基于第一性原理的白盒化路径。不被补偿场补偿的参数方向就是"物理自由度"——真正影响性能的参数。这为理解神经网络的"黑箱"提供了一种全新的方法论。

**对物理学的含义**：深度学习系统中的规范场结构不是人为设计的，而是从优化动力学中自然涌现的。这暗示规范场论可能不仅是物理学的理论框架，更是复杂信息处理系统的普遍组织原则。

### 6.4 与东方哲学的规范场论对应

| 易学概念 | 规范场论对应 | 实验证据 |
|---------|------------|---------|

| 简易 | 补偿场方程 | V91: gp重训恢复性能 |

"道可道，非常道"的规范场论解读：text_proj是规范可变的（"道可道"），但检索性能是规范不变的（"非常道"）。"道"不是某个特定的text_proj值，而是规范等价类。

### 6.5 局限性

1. **单一架构**：当前证据仅来自YiJingV51FourStage架构
2. **规范场的精确形式**：$\Delta: \delta\theta_c \mapsto \delta\theta_a$ 的精确数学形式尚未确定
3. **场强张量**：$\mathcal{F}_{ij} = \frac{\partial \theta_{a,i}}{\partial \theta_{c,j}} - \frac{\partial \theta_{a,j}}{\partial \theta_{c,i}}$ 是否为零尚未测量

## 7 结论

本文通过V34–V91的完整实验链，发现并验证了核心映射退化定理及其与规范场论的深刻平行：

1. **退化基态**：text_proj不是优化吸引子，而是退化基态——检索损失在text_proj方向上近乎平坦，存在多个等价解
2. **补偿场**：gua_prototype充当规范场，补偿text_proj的任何变化，维持检索性能的规范不变性
3. **规范场论结构**：深度学习系统中存在物质场（text_proj）+ 规范场（gua_prototype）→ 规范不变量（检索性能）的三层结构

这些发现将我们的论点从"发现了工程上的不动点现象"完全升级为"发现了深度学习模型中的退化基态及其与物理学规范场论的深刻平行"。这种平行不是模糊的哲学类比，而是可计算、可验证的物理实在——从Goldstone定理到规范不变性，从补偿场到守恒律，每一个概念都有精确的实验对应。

**道，规范可变；器，补偿而变；理，规范不变。**
